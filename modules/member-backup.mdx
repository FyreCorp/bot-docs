## Usage
You can easily restore a member by using `/restore @user`.
This command supports the following options:

- `user`: The member to restore.
- `keep-existing-roles`: If the member's current roles should be kept (default: No).


## Configuring
To configure the member backup module, use `/config member-backup`.
This command supports the following options:

- `enabled`: A true or false value if member backups are enabled.
- `expire-after`: The amount of hours member backups will expire after (max: 168, default: 24).
