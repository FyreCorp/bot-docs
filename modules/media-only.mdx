## Configuring
Configure Fyre to automatically delete messages that don't contain media in a specific channel.

To configure media-only mode in a channel, use `/channels media-only` and fill in the desired options.

### Config Options
There are a few options available for media-only mode, here's a list and what they do.

- `enabled`: If media-only is enabled in the target channel (default: No).
- `include-bots`: If messages from bots should be checked for media (default: No).
- `require-spoiler`: If all media should be required to be spoilered when posted (default: No).
